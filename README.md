# About The Project

This project is a web-based portfolio that showcases my photography work. The portfolio was built using a combination of powerful front-end technologies, including Tailwind, CSS, HTML, and JavaScript.

## Built With

- HTML
- CSS
- JavaScript (JQuery)
- Tailwind
